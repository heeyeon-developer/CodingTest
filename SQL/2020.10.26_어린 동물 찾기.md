><h1>프로그래머스_Level1_어린 동물 찾기</h1>
1. 모든 속성을 출력하기 위해 *을 활용
2. ANIMAL_ID의 오름차순으로 출력하기 위해 order by를 활용

```MySQL
select ANIMAL_ID, NAME
from ANIMAL_INS
where INTAKE_CONDITION != 'Aged'
order by ANIMAL_ID
```
이전에 풀었던 코드 - 동일
```MySQL
SELECT ANIMAL_ID, NAME
FROM ANIMAL_INS
WHERE INTAKE_CONDITION != 'Aged'
ORDER BY ANIMAL_ID ASC;
```
