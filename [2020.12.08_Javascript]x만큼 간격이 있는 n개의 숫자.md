><h1>프로그래머스_Level1_x만큼 간격이 있는 n개의 숫자</h1>
1. 시작값은 x를 초기값 start로 저장
2. x값만큼 연속 증가하므로 start에 계속 저장
3. n만큼 반복하며 answer에 push

```javascript
function solution(x, n) {
    var answer = [];
    let start = x
    for(let i=0; i<n; i++){
        answer.push(start)
        start+=x
    }
    return answer;
}
```
